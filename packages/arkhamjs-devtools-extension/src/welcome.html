Quick Setup

  1. Install the ArkhamJS DevTools middleware in your application
  2. Open Chrome DevTools (F12) and look for the "ArkhamJS" tab
  3. Start debugging your ArkhamJS application!

Installation Example:

npm install @nlabs/arkhamjs-middleware-devtools import { Flux } from '@nlabs/arkhamjs'; import { DevTools } from '@nlabs/arkhamjs-middleware-devtools'; Flux.init({ name: 'my-app', stores: [UserStore, CartStore], middleware: [ DevTools({ enableTimeTravel: true, enableStateDiff: true, logPerformance: true }) ] });
